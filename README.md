## Installation

1. clone project locally

    ```sh
    git clone 'https://github.com/boite-informatique/startup-front'
    ```

2. go to project directory

    ```sh
    cd startup-front/
    ```

3. Install dependencies

    ```sh
    npm install
    ```

4. Start the development server

    ```sh
    npm run dev
    ```
